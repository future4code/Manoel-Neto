Monitoramento de Abate

Desenvolvido como parte do projeto de final de curso do National College of Ireland, o software Monitoramento de Abate foi criado para um negócio familiar, um frigorífico. O qual, desde quando, foi inaugurado utiliza da mesma forma, diga-se de passagem, obsoleta para fazer o monitoramento e pesagem do produto (carnes). A famosa caneta e papel. Com o intuito de atualizar a empresa, deixando-a mais inserida no mercado tecnológico, surgiu a ideia de desenvolver este software. O qual facilitaria o processo e o tornaria ao mesmo tempo, mais otimizado e tecnológico. Poupando tempo para a emissão de relatórios, calculo de pagamento para os vendedores da matéria prima (gado), e tornando possivel o arquivamento dos mesmos em um banco de dados para o próprio controle da empresa, sendo possivel acessar os mesmos quando necessário.

Por se tratar de um software um pouco complexo para um, antes, Engenheiro Civil totalmente leigo quanto a programação e devido ao pouco tempo para desenvolvimento do mesmo, não foi possível o término do mesmo. A idéia inicial era bastante simples, mas que durante a execução do software novas ideias foram surgindo e assim deixando o projeto cada vez mais complexo. Inicialmente, seria desenvolvido com as funções básicas, mais conhecidas como CRUD, que são: Create (criar), Read (ler), Update(atualizar), Delete (deletar). Onde o administrador poderia cadastrar outros administradores e também os vendedores. Também, dar início ao processo de abate, o qual permitia selecionar o vendedor dono dos animais, o admnistrador realizando o processo e conforme o produto fosse pasando pela balança inserindo os pesos.

Quando o processo fosse terminado, um relatório onde estaria disponivel todos os pesos, juntamente com o preço pago pela empresa e o total final que seria pago ao consumidor seria impresso. No meio do desenvolvimento do projeto veio a idéia de desenvolver um site com uma API consumidora, onde os produtores teriam acesso abate e ao seu histórico de abates, o qual poderia ser acessado pelo mesmo sempre que necessário ao invés de precisar ir sempre na empresa para retirar os mesmos.

Tecnologias Usadas: Java, Web Services API, HTML, CSS e MySQL.
