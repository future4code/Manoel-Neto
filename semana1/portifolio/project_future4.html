Future4 - Labenu

Desenvolvido como parte do processo de seleção para os cursos, da antiga Future4, atualmente Labenu. Foi pedido aos candidatos, alguns já com conhecimento prévio e outros sem total conhecimento, que ao olhar uma imagem fornecida pela empresa, desenvolvessem o código para que a imagem fosse replicada em website. Fornecendo ao candidatos um grupo, onde os mesmos podiam interagir entre si para que uns ajudassem aos outros, observando também, questões comportamentais dos candidatos. Porém, não poderia haver o compartilhamento de códigos muito menos a cópia, o projeto deveria ser totalmente autoral.

Tecnologias Usadas: HTML, CSS e JavaScript.
